FIX 503 ERROR — api.aljaroshi.com NOT RUNNING
=============================================

Your .env is OK. The problem is: Node.js app is NOT started on the SERVER.
Do NOT run commands in Windows PowerShell — use cPanel Terminal on the server.

STEP 1 — cPanel > Setup Node.js App
-----------------------------------
Create or EDIT the app with these EXACT settings:

  Node.js version:     18 or 20 (LTS)
  Application mode:    Production
  Application root:    /home/aljafvrp/api.aljaroshi.com
  Application URL:     api.aljaroshi.com
  Application startup: server.js          <-- IMPORTANT (not dist/index.js)

  Environment variables (add in the UI OR use .env file):
    DATABASE_URL = mysql://aljafvrp_crmuser:...@localhost:3306/aljafvrp_crmdatabase
    JWT_SECRET   = AlJaroshi-ServiceOS-2026-UseLongRandomSecretHere
    FRONTEND_URL = https://crm.aljaroshi.com
    NODE_ENV     = production

  NOTE: cPanel shows a PORT number (e.g. 5000 or 34567).
  - Put that SAME port in your .env:  PORT=5000
  - Copy the .htaccess snippet from cPanel UI (it has the correct port)
  - Replace your .htaccess with cPanel's version if ports don't match

STEP 2 — cPanel > Terminal (on SERVER, not your PC!)
----------------------------------------------------
  cd /home/aljafvrp/api.aljaroshi.com
  ls -la

  You MUST see: server.js  package.json  dist/  prisma/  .env

  npm install
  npx prisma generate
  npm start

  If "Server running on port..." appears → app works. Press Ctrl+C then do Step 3.

STEP 3 — Restart in Setup Node.js App
--------------------------------------
  Click STOP APP → then START APP (or RESTART)

STEP 4 — Test
-------------
  Browser: https://api.aljaroshi.com/health
  Must show: {"status":"ok","message":"Server is healthy"}

  Then login: https://crm.aljaroshi.com/login

COMMON MISTAKES
---------------
  X Running "cd /home/aljafvrp/..." in Windows PowerShell (does NOT work)
  X Only uploading zip without "npm install" on server
  X Wrong startup file (use server.js)
  X .htaccess port 5000 but cPanel app uses different port → 503
  X Missing dist/ folder after upload

IF npm start SHOWS ERROR — copy the red error text and send screenshot.
