NO TERMINAL — cPanel Only Fix (api.aljaroshi.com)
==================================================

PART A — Upload backend files
-----------------------------
1. PC se file kholo: backend-cpanel-deploy.zip
2. cPanel > File Manager
3. Folder: api.aljaroshi.com (ya api.aljaros — jo API folder hai)
4. Purani files backup (optional): select all > Compress > Download
5. Upload > backend-cpanel-deploy.zip
6. Right-click zip > Extract
7. Confirm ye files dikhen:
   - server.js
   - package.json
   - dist/ (folder)
   - prisma/ (folder)
   - .htaccess

PART B — .env file (File Manager)
----------------------------------
1. Same folder mein .env file (aapki wali — save karo):

DATABASE_URL="mysql://aljafvrp_crmuser:database%401234@localhost:3306/aljafvrp_crmdatabase"
JWT_SECRET="AlJaroshi-ServiceOS-2026-UseLongRandomSecretHere"
PORT=5000
FRONTEND_URL="https://crm.aljaroshi.com"
NODE_ENV=production

PART C — Setup Node.js App (MAIN FIX — no terminal)
---------------------------------------------------
1. cPanel home > search: "Setup Node.js App" (ya "Node.js Selector")
2. Agar purani app hai > EDIT. Nahi hai > CREATE APPLICATION

3. Fill EXACTLY:

   Node.js version:        18.x ya 20.x (sab se latest LTS)
   Application mode:       Production
   Application root:       api.aljaroshi.com
                           (full path: /home/aljafvrp/api.aljaroshi.com)
   Application URL:        api.aljaroshi.com
   Application startup:    server.js

4. Environment variables — yahan bhi add karo (cPanel UI mein + button):

   Name: DATABASE_URL
   Value: mysql://aljafvrp_crmuser:database%401234@localhost:3306/aljafvrp_crmdatabase

   Name: JWT_SECRET
   Value: AlJaroshi-ServiceOS-2026-UseLongRandomSecretHere

   Name: FRONTEND_URL
   Value: https://crm.aljaroshi.com

   Name: NODE_ENV
   Value: production

   Name: PORT
   Value: 5000

5. SAVE

6. Same page par button dikhega:
   "Run NPM Install"  — IS PAR CLICK KARO (1-3 min wait)
   (Yeh terminal ki jagah hai — packages install karta hai)

7. Phir button:
   "Restart" ya "START APP" — CLICK

PART D — .htaccess port match
-------------------------------
Setup Node.js App page par neeche "Apache configuration" ya
".htaccess code" dikhta hai — usme port number hota hai.

Agar wahan 5000 nahi hai (jaise 34567):
  - File Manager > .htaccess > Edit
  - 127.0.0.1:5000 ko badlo us port se jo cPanel dikhata hai
  - .env mein PORT bhi wahi number lagao
  - Node.js App > Restart

PART E — Database SQL (phpMyAdmin — no terminal)
------------------------------------------------
1. cPanel > phpMyAdmin
2. Left: aljafvrp_crmdatabase
3. Tab: SQL
4. File production-cpanel-update.sql ka content paste karo > Go
   (Duplicate column errors = OK, skip)

PART G — Agar ab bhi 503
------------------------
Setup Node.js App page par "Open Application" ya logs dekho.

Checklist:
 [ ] server.js exists in api folder
 [ ] dist/ folder exists (empty nahi)
 [ ] "Run NPM Install" click hua
 [ ] App RESTART hua (STOP phir START)
 [ ] .htaccess port = cPanel port

Agar "Run NPM Install" button NAHI hai:
  Hosting support ko likho: "Please run npm install in /home/aljafvrp/api.aljaroshi.com"
  Ya Live Chat — 2 minute ka kaam hai unke liye.
